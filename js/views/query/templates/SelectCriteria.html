Researching a patent is part of the task of getting a new invention to market. The PatentsView Query Tool is a powerful search tool that longitudinally links inventors, their organizations, locations, and overall patenting activity, returning a customized data set by e-mail.

For researchers, in particular, the query tool enables a broad spectrum of users to examine the dynamics of inventor patenting activity over time and space. Users can use the query tool to explore patent technologies, assignees, citation patterns, and co-inventor networks.

Preliminary patent searches can be very beneficial for researchers and inventors. They help avoid submitting a patent application with claims already described. Also, specifically for inventors, they prepare you to discuss your ideas with a registered patent attorney or patent agent.

How to generate a customized data set with the PatentsView query tool:

  1. For a QUICK SEARCH query tool: Input one or more descriptive terms (separated by spaces) into the search bar above, and click
    . The query tool will search the PatentsView database for patents where each term appears in the patent title or abstract.
    For an Advanced Search query tool: Click “Advanced Search >”. On the next page, select your category of interest: patents, inventors, or assignees. Query results will be organized based on these primary categories, which correspond to the PatentsView API Endpoints. Then, create one or more search criteria by choosing a search field, selecting from the available options in the dropdown menus, entering your search term, and clicking “Add to Search.” When a search criterion is added, it will appear in the Search Summary box, where you also can remove unwanted criteria. Once all desired search criteria are added to the Search Summary, click “Submit Search.”
  2. On the next page, you can customize the information included in your output data by checking or unchecking any of the fields listed. Fields are grouped into five categories, which can be expanded by clicking on the plus sign next to each category name. Note that selecting multiple many-to-one output fields (e.g., inventor first and last name and assignee organization name for a patent query) can result in duplicate rows in the output data if the CSV format is selected.
  3. Click “Preview Results” to view the next screen, where query tool results can be previewed in CSV or JSON format. If you get zero results from your query, consider fewer or alternative search terms.
  4. Below the preview pane, you can sort your results by selecting a group and field from the dropdown menus. Then, specify an e-mail address for delivery and select your preferred file format(s).
  5. Finally, click “Submit Query.” Upon completion of a successful query, results will be sent to the designated e-mail address.
